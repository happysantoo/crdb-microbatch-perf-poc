## Markdown Image Embedding Guide

### Basic Syntax:
```markdown
![Alt Text](path/to/image.png)
```

### Relative Paths:
- From `documents/analysis/`: Use `../images/grafana-screenshots/filename.png`
- From root: Use `documents/images/grafana-screenshots/filename.png`

### Example:
```markdown
![Dashboard Overview](../images/grafana-screenshots/dashboard-overview.png)
*Caption: Full Grafana dashboard during load test*
```

### Tips:
1. Use descriptive alt text (for accessibility)
2. Add captions below images
3. Use PNG format for graphs/charts
4. Keep file sizes reasonable (<2MB)
